# Recursive Harness Self-Improvement — bounded reproduction

This public repository contains a clean-room, open-model reproduction protocol for arXiv
2607.15524. Eight deterministic repository-building tasks, hidden executable tests, blinded
reversed-order judging, prompt-harness revisions, and token/context accounting are implemented
under `reproduce/`. Formal results, figures, and the tutorial notebook will be added after the
Kubernetes experiments complete.

The fixed experiment command is:

```bash
bash run.sh
```
